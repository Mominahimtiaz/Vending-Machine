 Smart Vending Machine python

A console-based vending machine application developed in Python that simulates a real vending machine experience.

 Project Overview
This program allows users to:
- Select drinks and snacks using item codes
- Insert money with validation
- Receive correct change
- Make multiple purchases in one session
- View intelligent product suggestions

 Features
- Categorized Menu: Drinks and snacks sections
- Unique Item Codes: Easy product selection (e.g., A1, B3)
- Stock Management: Tracks available inventory
- Money Validation: Accepts coins and bills, returns change
- Multiple Purchases: Continue shopping or exit
- Smart Suggestions: Recommends products based on selection
